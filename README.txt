Para compilar de una manera correcta y sin errores seguir los siguientes pasos:

Tener todos los archivos dentro de una misma carpeta, en el caso del teclado.cpp de no ser así modificar codigo para poder leer el archivo teclado-entrada.txt desde donde se encuentren.

Luego en la terminal de bash acceder a la carpeta y buscar el archivo o buscar el archivo donde se encuentre en caso de ser necesario.

Cuando se encuentre en la misma direción que el archivo ejecutar el siguiente codigo.

Para el codigo del teclado:

	Para compilar: g++ teclado.cpp -o teclado -Wall
	Para ejecutar: ./teclado

Para el codigo del ranked-abb:

	Para compilar: g++ ranked-abb.cpp -o ranked-abb -Wall
	Para ejecutar: ./ranked-abb

La forma de ejecución usada es mediante g++, siguiendo los pasos de arriba.
